# Personal Portfolio Wesbite

The code for me personal website, where I share some of my current coding projects, writing, etc..

View the live website at <https://huntercbuxton.com>.

## Tech Stack

- Node.js (v20+)
- React.js
- JSON (local storage)

## Quick Start

### Installation

run ```./install.sh <release_branch_name>```in project root. If no branch name is passed as an argument, the main branch will be installed by default.

### Rehydration

run ```./rehydrate.sh``` in project root
